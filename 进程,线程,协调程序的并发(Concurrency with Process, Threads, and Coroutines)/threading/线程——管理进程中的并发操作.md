目标：管理多个执行线程
使用多线程可以使程序在一个进程空间内运行并发地执行多个操作。
##线程对象
使用一个线程最简单的方式就是用一个目标函数去实例化它并调用start()函数让它开始工作。
```python
# threading_simple.py
import threading


def worker():
    """thread worker function"""
    print('Worker')


threads = []
for i in range(5):
    t = threading.Thread(target=worker)
    threads.append(t)
    t.start()
```
上面程序将会输入五行"Worker"。
```bash
$ python3 threading_simple.py

Worker
Worker
Worker
Worker
Worker
```
